# Banking-Risk-Analytics-Project

📊 Banking Risk Analytics Project

🔹 Project Overview :
- This project focuses on analyzing banking customer data to understand customer financial behavior, lending patterns, deposit trends, and risk-related insights using Python, SQL, and Power BI.
- The project was completed in multiple phases, starting from Exploratory Data Analysis (EDA) and progressing toward interactive dashboard development and detailed business analysis.

🎯 Project Objective :
- The objective of this Banking Risk Analytics project is to analyze customer financial and banking data to identify meaningful insights related to loans, deposits, customer engagement, and banking performance.
  
This project aims to:
- Understand customer financial behavior through data analysis
- Analyze lending and deposit patterns across different customer segments
- Identify trends based on income band, nationality, occupation, and banking relationship
- Perform risk-related analysis using customer financial indicators
- Develop interactive dashboards for business reporting and decision-making
- Transform raw banking data into actionable business insights using Python, SQL, and Power BI
  
The project demonstrates how data analytics and business intelligence techniques can be applied in the banking domain to support data-driven decision-making and performance analysis.

🔹 Phase 1 – Python EDA & Data Understanding
- In the first phase, the focus was on understanding the banking dataset and preparing the data for analysis.
  
Key Tasks Performed:
- Loaded and inspected the banking dataset using Pandas
- Performed data cleaning and preprocessing
- Checked missing values, data types, and summary statistics
- Created Income Band segmentation (Low, Mid, High)
  
Conducted univariate analysis on:
- Estimated Income, Occupation, Nationality, Risk Weighting, Fee Structure
  
Skills Applied:
- Exploratory Data Analysis (EDA), Data Cleaning, Feature Engineering, Data Visualization

🔹 Phase 2 – Power BI Dashboard Development
- In the second phase, interactive dashboards were created in Power BI to analyze banking KPIs and customer financial behavior.
  
Dashboards Developed:
- Home Page Dashboard
- Loan Analysis Dashboard
- Deposit Analysis Dashboard
- Banking Summary Dashboard
  
Key Insights:
- Loan and deposit distribution analysis, Banking relationship analysis, Nationality-wise financial trends, Occupation-based loan analysis, Customer engagement analysis

Features Added:
- Dynamic slicers and filters, KPI cards, Interactive visualizations, Drill-down analysis

Business Value:

The project helps identify patterns in customer financial behavior, lending exposure, and banking engagement to support better business understanding and data-driven decision-making.
